Contents of the fundamentals

  1. HTML Document

    2. Made up of the trio

    Elements and attributes

    Doctype Declaration - Placed before html tag to declare version of the html

  2. HTML Structural Elements

    3. Mainly referring to head and html

    Parent and child relations

    Metadata - Data that describes other data

    Meta attribute - such as Description, keywords and author for use by third party (search engines for example)

  3. HTML Content Elements

    4. Mainly referring to body

    Elements - Define structure and content via tags (<> and )

  4. Headings

    5. h1 to h6

    Bold by default, only have one h1 per page

  5. Paragraphs

    6. Block level elements

  6. Div and Spans

    7. For semantically subjective elements

    div cannot go into span but

    Can contain any content and context

    It is semantically neutral. AKA Structural elements

    Block level elements - occupies entier space of its container (A block)(div)

    Inline level elements - smaller chunks that take up space based on width and height (span)

  7. Comments

  8. Attributes

    9. Property of an element

    class most common styling property and semantically grouping for an element

    id is a property, can only have one (best not to use it)

  9. Links

    10. Anchor elements

  10. Lists - Ordered and Unordered

    11. Use type to determine type of numbering/bulletpoint

    Ordered - i, ii, iii, iv etc

    Unordered - *, *, * etc

  11. Lists - Description and Nested

    12. Description list declared by dt and dd is definition, dd is optional and can have multiple

    Nesting - put something inside another

  12. Images

    13. img element with src and alt attributes

  13. Forms

    14. Used to exchange data between user and application

    Action (Location to query) and method (GET or POST) attribute most important

    HTTP (Hypertext Transfer Protocol)

  14. Form Description

    15. The child elements that can be in a form

    Input (type, id, placeholder, name[important], value attributes) element most used

    Label (for attribute) to designate label to an input's id attribute

    Better to put into placeholder instead of a label

    Input is a self closing element (no need closing tags)

    Other input type attributes - color, file, number, reset, search and range

  15. Form Actions

    16. More form/input type attributes

    checkbox - choose at least 1 of the list

    radio - choose only 1 of the list

    select (name) element - nested in form element, to enable a dropdown menu

    option (value) element - nested in select element, provides options for dropdown menu

    submit - sends all form data, can be used like button element

  16. Form Semantics and Validation

    17. Provides semantics and validation by grouping stuff

    fieldset element - grouping of several form children

    legend element - usually followed by fieldset, to give a description for that field

    optfield element - used inside select element, to group options together

    required attribute - used to ensure certain fields are filled up, like username and password

  17. IFrames

    18. iframe element - used to embed another web document (such as a web page), only has src as an attribute

  18. Semantic Web

    19. HTML5 introduces more semantic elements

    header, nav, main, section, article, aside, footer
