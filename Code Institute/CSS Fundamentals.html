Contents of the fundamentals

  1. CSS Basics

    2. Alters and decorates HTML

    a(selector) {background-color(property):yellow(value);}(declaration)

  2. CSS Type & Class Selectors

    3. Type selector i.e p, h1 etc

    Class selector i.e class="jumbotron" / .jumbotron

  3. CSS ID and Attribute Selectors

    4. ID selector i.e id="unique" / #unique

    Generally speaking, dont use ID

    Attribute selector i.e input name="name" / input[name]

  4. CSS DOM Selectors

    5. Direct Child selector i.e div then span / div > span

    Descendants of selector i.e div then span / div span

    Declare specific first, general later

    Multiple descendants of selector i.e div then span then ol / div span ol

    Sibling selector i.e div next to span / div ~ span

  5. CSS - Inline and Embedded

    6. Inline CSS - often used for debugging because fast, can slow dev process because needs to be constantly implemented

    i.e div style="border: 1px solid #4a4a4f;"

    Embedded CSS - in head element, not great but ok for styling that page

    i.e in head element, style element with style declrations inside targetting specific areas of the html doc

  6. External CSS and precedence

    7. External CSS - external css file linked to html via link element inside head

    i.e link rel="stylesheet" href="location of css.css" type="text/css"

    Best as can be minifed, recycled, faster loading and seperation of concerns

    Precendence of css styles = inline > embedded > external

    !important placed after value - forces that css to always be active/highest precedence (dont use this)

  7. CSS Rules

    8. Inheritence - child will inherit everything from parent except what is declared

    Can use inherit as a value on any CSS attribute

  8. Comments and font-family

    9. /* like this */

    Font family can be imported, when declared need to assign fallback default

  9. Working with Text

    10. Measurement of units - Relative and Absolute

    Relative depends on relation to object

    Pixels - px

    Em - em size = pixel size/16 or 16px / based on font size usually 16px

    Em can sometimes cause design issues as multiple inheritence will alter it's default size

    Rem - em but are consistent, does not inherit from parents

    Percentage - %

    Viewpoint width - vw / percentage of your viewport, adjusts accordingly

    Viewpoint height - vh / same as above but for height

  10. Font Parameters

    11. Font size - key values (medium, x-small, xx-small) or px

    Font Weight - key values (bold, bolder, lighter) or 100-900 range

  11. Font Color

    12. Color - Hexadecimal (#bada55), regb(255, 255, 255), keyword (blue, green, red)

  12. Font Position

    13. text-align or vertical-align

    text-shadow - shadow in horizontal direction, distance in vertical direction, blur effect and color

  13. Font Spacing

    14. word-spacing

    word-wrap - break-word or normal

    word-break - break-all, keep-all or normal

  14. Letter Spacing

    15. letter-spacing - usually used on site logos and headings

    White-space - pre-wrap, nowrap, pre or pre-line

  15. Font Style

    16. text-transform - capitalize, uppercase or lowercase

    text-decoration - underline, overline, line-through or none

    font-style - italic or normal

  16. The Box Model

    17. Starting from outmost - Margin, Border, Padding (Content is the innermost untouched box)

    padding - creates space between content and border

  17. Box Border and Margin

    18. border - creates space between padding and margin

    margin - creates space around elements

  18. Background Appearance

    19. background-color - affects padding area

    background-image - url('location'), can have multiple

    background-repeat - repeat, repeat-y, repeat-x or no-repeat

  19. Background Positioning

    20. background-size - contain, cover or measurement units (px, em, vw etc)

    background-position - left, right, top, bottom or center (also can combine like top right)

    Can also use measurement units for X and Y such as 16px 16px

    background-clip - which part of the box model does background extend to

  20. Background Source

    21. background-attachment - fixed or scroll / tells browser behaviour of bg

    background-origin - where in box model bg begins

    shorthand = bg-image bg-position / bg-size repeat-style bg-attachment bg-origin bg-clip bg-color

  21. Box Sizing

    22. box-sizing - border-box / controls if border and padding is included in size of element

  22. Layout

    23. display - block, inline or inline-block

    div - Block level elements

    span - inline elements

  23. Visibility

    24. visibility - hidden(still there just invisible) or none (not there)

  24. Positioning

    25. position - static(default)

    position - relative(movement based on it's location statically)

    position - absolute(not part of page flow, positioning based of closest ancestors, otherwise initial block)

    position - fixed(not part of page flow, will always be displayed and is not moved

  25. Z-Index

    26. z-index - number / controls the stacking order of elements (highest and latest declared is strongest)

  26. Layout Interactions

    27. overflow - auto(scrollbar), scroll(either x or y), hidden or visible

    overflow-x and overflow-y is same as above but only for either or

    float - left or right / position element beside another element like microsoft word

    clear - left, right or both / related to float, whether elements can be put next to it

    Horizontal centering - auto left and right margin will centre any block level element

    Centering inline-block elements - add text-align to the parent

    Relative positioning is same as above

    Absolute positioning elements we need to set left and right auto also

    Vertical centering - transform translateY
