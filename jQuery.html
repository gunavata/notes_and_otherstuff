What is jQuery?

Javascript library Simplifies javascript programming HTML/DOM manipulation Effects and animations AJAX Utilities

Basic Syntax

$(selector).action()

  • $ = define/acces jQuery
  • (selector) = query or find HTML elements
  • action() = what to do on element
  • EX: $(document).ready(function() {jQuery methods here});

Selectors work similiarly to CSS selectors with a few additions

  • $("*") = Select all element
  • this = select current HTML
  • [attr] = select all elements with that attribute
  • tr:even + tr:odd = select all even or odd tr elements

It works on events (Firing or events i.e keypress event is fired)

on() method attaches one or more event handlers to selected element

Some Effects

  • hide(speed, cb)
  • toggle(speed, cb) = hide or unhide
  • fadeIn/Out(spd, cb)
  • fadeToggle(spd,cb) = hide or unhide
  • fadeTo(spd, opacity, cb)
  • slideDown/Up(spd, cb)
  • slideToggle(spd, cb) = hide or unhide
  • animate({params}, spd, cb)

    • params include what properties to alter i.e left: 250px

    params can include multiple properties i.e left: '250px', opacity: '0.5'

    Relative values = += or +- i.e height: '+=150px';

    Pre definied values = specify property animation value i.e height: 'togglt'

    Follows queue functionality

  • stop(stopAll, goToEnd) = both optional, stopAll cancels all animation, goToEnd specifies if finish animation first or not
  • Callback functions = waits for code to finish before proceed i.e hide("slow", function(){})
  • Method Chaining = $(element).css(code).slideUp(slow).slideDown(fast)

DOM Manipulation

Get and Set Content

  • text(content) = sets or returns text content of selected element
  • html(content) = sets or returns content of selected elements with HTML markup
  • val(content) = sets or returns value of form fields
  • attr(target, what to change) = req + opt / sets or return attribute values

    • The above elements can accept cb i.e text(function(index of current element, original value)

  • append(content) = insert content at the end
  • prepend(content) = insert content at beginning
  • after(content) = insert content after selected element
  • before(content) = insert content before selected element

    • Selected content is put before i.e $("img").after()

    All four methods above can take multiple elements in order to chain i.e append(txt1, tx2, txt3)

  • remove(filter) = filter rules which to remove / removes selected and child
  • empty() = removes only child
  • addClass(className) = adds one or more class to selected elements
  • removeClass(className) = removes one or more class from selected elements
  • toggleClass(className) = toggles between adding or removing class
  • css("propertyname", "value(if setting") = sets or returns style attribute

Traversing within the DOM

Move through/find/select html based on their relation

Start with selector and move through DOM tree via ancestors, descendants, siblings or current element

  • Going upwards
    • parent() = direct parent
    • parents(filter) = filter out only these / all parents
    • parentsUntil(target) = parents between selector and target
  • Going Downwards
    • children(filter) = filter sets a criteria / all direct children, fetches all of them
    • find() = look for all descendant elements down to the last
  • Going Sideways
    • siblings(filter) = filter targes specific / returns all siblings
    • next() = returns next sibling
    • nextAll() = returns all next siblings
    • nextUntil(target) = returns all next siblings until target
    • prev()
    • prevAll()
    • prevUntil()
  • Filtering
    • first() = selects first found selector
    • last() = selects last found selector
    • eq(index) = selects index number selector
    • filter(target) = selects all true selector
    • not(target) = opposite of filter()
